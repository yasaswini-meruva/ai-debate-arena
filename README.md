# AI Debate Arena

AI Debate Arena is a Multi-Agent AI Debate Platform where two AI agents debate a topic using Retrieval-Augmented Generation (RAG), document evidence, voice narration, and automated AI judge scoring.

## Features

* Multi-Agent AI Debate System
* Retrieval-Augmented Generation (RAG)
* PDF Knowledge Base Upload
* Document-Based Evidence Retrieval
* AI-Powered Fact Support
* Voice Narration for Arguments
* Automated Judge Evaluation
* Futuristic Interactive UI
* Live Web Deployment

## Tech Stack

### Frontend

* React

* CSS

### Backend

### AI Components

* LLM-Based Debate Agents
* RAG Engine
* ChromaDB Vector Database
* Voice Synthesis
* AI Judge Evaluation System

## How It Works

1. Upload a PDF document.
2. Enable RAG mode.
3. Enter a debate topic.
4. AI Alpha and AI Omega debate using retrieved document evidence.
5. Judge AI evaluates logic, evidence quality, and rebuttal strength.
6. Voice AI narrates arguments and verdict.
