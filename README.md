# AI Debate Arena

AI Debate Arena is a Multi-Agent AI Debate Platform that enables two AI agents to debate a topic using Retrieval-Augmented Generation (RAG), Voice AI, and Automated Judge Scoring.

## Features

- Multi-Agent AI Debate System
- Retrieval-Augmented Generation (RAG)
- PDF Knowledge Base Upload
- Evidence Verification
- Voice Narration for AI Arguments
- AI Judge Scoring
- Interactive Futuristic UI

## Tech Stack

### Frontend
- React
- Vite
- CSS

### Backend
- FastAPI
- Python
- Groq API

### AI Components
- LLM-based Debate Agents
- RAG Engine
- Voice Synthesis
- Judge Evaluation System

## How It Works

1. Upload a PDF document.
2. Enable RAG mode.
3. Enter a debate topic.
4. AI Alpha and AI Omega debate using document evidence.
5. Judge AI evaluates logic, evidence quality, and rebuttal strength.
6. Voice AI narrates arguments and verdict.
